## 关键发现

### 🎉 ThingsBoard 支持 TBEL 脚本

ThingsBoard 的 Calculated Field 支持 **TBEL (ThingsBoard Expression Language)**，这是完整的 JavaScript 语法支持！

#### TBEL 核心能力
```javascript
function expression(ctx, arg1, arg2, ...): boolean {
  // 完整的 JavaScript 语法
  // 可访问参数值和时间戳
  // 支持复杂逻辑判断
}
```

#### ctx 对象结构
```javascript
{
  "temperature": {
    "ts": 1740644636669,  // 时间戳
    "value": 98.7         // 值
  },
  "latestTs": 1740644636669
}
```

## TBEL 脚本示例

### 示例 1：简单条件
```javascript
function expression(ctx, temperature) {
  return temperature > 80;
}
```

### 示例 2：持续时间判断
```javascript
function expression(ctx, temperature) {
  var currentTs = ctx.latestTs;
  var temperatureTs = ctx.args.temperature.ts;
  var durationSeconds = (currentTs - temperatureTs) / 1000;
  
  // 温度超过 80 度，且持续 300 秒（5分钟）
  return temperature > 80 && durationSeconds >= 300;
}
```

### 示例 3：多条件组合
```javascript
function expression(ctx, temperature, pressure) {
  // 独立触发：任一条件满足
  return temperature > 80 || pressure < 20;
}
```

### 示例 4：联动触发
```javascript
function expression(ctx, temperature, pressure, humidity) {
  // 独立触发
  var condition1 = temperature > 80;
  var condition2 = pressure < 20;
  
  // 联动触发：多个条件同时满足
  var condition3 = temperature > 70 && pressure > 100 && humidity > 60;
  
  return condition1 || condition2 || condition3;
}
```

### 示例 5：时间范围判断
```javascript
function expression(ctx, temperature) {
  var currentTs = ctx.latestTs;
  var hour = new Date(currentTs).getHours();
  
  // 温度超过 80 度，且在 8:00-18:00 之间
  return temperature > 80 && hour >= 8 && hour < 18;
}
```

### 示例 6：多级别条件
```javascript
function expression(ctx, temperature) {
  // CRITICAL: > 90
  if (temperature > 90) return true;
  
  // MAJOR: > 80
  if (temperature > 80) return true;
  
  // MINOR: > 70
  if (temperature > 70) return true;
  
  return false;
}
```

### 剩余风险

#### 🟡 中风险：API 限流
- **缓解**：限流控制（50 次/秒）+ 失败重试

#### 🟡 中风险：外部接口不稳定
- **缓解**：缓存机制 + 降级策略

#### 🟢 低风险：数据一致性
- **缓解**：定时同步 + 状态监控
