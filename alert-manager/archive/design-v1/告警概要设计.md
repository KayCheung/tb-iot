## 二、Topic说明 
| Topic 名称             | MQ/MQTT | 生产者                     | 消费者                     | 使用场景                                   | 作用                                                                 |
|------------------------|---------|----------------------------|----------------------------|--------------------------------------------|----------------------------------------------------------------------|
| `alarm.trigger-event`  | Kafka   | TB规则链                   | 告警核心                   | 规则条件满足后的触发事件                   | TB携带具体触发级别和值、元数据等信息                                 |
| `alarm.lifecycle-event`| Kafka   | 告警核心                   | 通知规则匹配器             | 告警状态变化全事件（ACTIVE→CONFIRMED→RESOLVED→CLOSE） | 统一告警事件总线，所有下游系统订阅                       |
| `notify.send`          | Kafka   | 通知分发器                 | 渠道发送器（SMS/Email/IM） | 通知任务异步发送队列                       | 发送解耦+重试，每用户每渠道一条记录                                 |

## 三、时序图


```
sequenceDiagram
  participant UI as 告警规则管理UI
  participant RuleSvc as 规则管理服务
  participant SyncSvc as TB同步服务
  participant TB as ThingsBoard(API+RuleEngine)
  participant Kafka as Kafka
  participant AlarmCore as 告警核心服务
  participant StationSvc as 测站/设备服务
  participant MQ as 告警MQ(通知)

  UI->>RuleSvc: 创建/更新告警规则(含级别/联动配置/时间窗口)
  RuleSvc->>RuleSvc: 写入本地告警规则表
  RuleSvc->>SyncSvc: 规则同步任务(规则ID列表)
  SyncSvc->>TB: 调用 TB API 创建/更新告警规则(按 Device)
  TB-->>SyncSvc: 返回 TB 规则ID/状态
  SyncSvc->>RuleSvc: 更新本地规则与 TB 规则ID映射

  TB->>TB: 设备D1遥测触发内部告警
  TB-->>Kafka: 发送告警事件(创建/更新)

  AlarmCore->>Kafka: 订阅告警Topic
  Kafka-->>AlarmCore: 告警消息(tbAlarmMsg)

  AlarmCore->>AlarmCore: 解析tbAlarmMsg(设备ID、告警类型、级别等)
  AlarmCore->>StationSvc: 查询设备-测站映射
  StationSvc-->>AlarmCore: 返回测站列表 + 设备信息

  AlarmCore->>RuleSvc: 获取对应本地告警规则(按设备/测站/类型)
  RuleSvc-->>AlarmCore: 返回规则及联动配置

  AlarmCore->>AlarmCore: 判断时间窗口、独立/联动触发
  AlarmCore->>AlarmCore: 生成平台告警实例、更新状态(本地库)

  AlarmCore->>MQ: 发送告警MQ消息(告警ID/测站/设备/级别/联动详情)
  MQ-->>下游通知模块: 通知模块订阅处理
```


## 四、接口说明

### 4.1 查询告警列表 

接口说明：告警模块 - 告警查询，支持多维度筛选和分页 
接口地址：`/api/v1/alarms` 
请求类型：`GET` 
接口入参： 
| 字段名       | 类型    | 是否必填 | 描述             | 示例值                     |
|--------------|---------|----------|------------------|----------------------------|
| `stationId`  | long    | 否       | 测站ID过滤       | `1001`                     |
| `deviceId`   | long    | 否       | 设备ID过滤       | `3001`                     |
| `ruleId`     | long    | 否       | 规则ID过滤       | `1001`                     |
| `level`      | string  | 否       | 告警级别         | `CRITICAL`                 |
| `status`     | string  | 否       | 告警状态         | `ACTIVE`                   |
| `startTime`  | string  | 否       | 开始时间         | `2026-02-01T00:00:00Z`     |
| `endTime`    | string  | 否       | 结束时间         | `2026-02-01T23:59:59Z`     |
| `keyword`    | string  | 否       | 关键字           | `水位`                     |
| `page`       | integer | 否       | 页码，默认1      | `1`                        |
| `size`       | integer | 否       | 每页数量，默认20 | `20`                       |
| `sort`       | string  | 否       | 排序字段,desc/asc| `firstTriggerTime,desc`    |

接口出参：
| 字段名                          | 类型    | 是否必需 | 描述               | 示例值                     |
|----------------------------------|---------|----------|--------------------|----------------------------|
| `code`                          | string  | 是       | 业务状态码         | `200`                      |
| `message`                       | string  | 是       | 异常消息           | `服务器异常`               |
| `data.page`                     | integer | 是       | 当前页码           | `1`                        |
| `data.pages`                    | integer | 是       | 总页数             | `20`                       |
| `data.total`                    | integer | 是       | 总记录数           | `200`                      |
| `data.items[].id`               | long    | 是       | 告警ID             | `10001`                    |
| `data.items[].ruleName`         | string  | 是       | 规则名称           | `泵站高温规则`             |
| `data.items[].deviceName`       | string  | 是       | 设备名称           | `泵站A-1#水泵`             |
| `data.items[].stationName`      | string  | 是       | 测站名称           | `泵站A`                    |
| `data.items[].metricName`       | string  | 是       | 指标名称           | `水位`                     |
| `data.items[].level`            | string  | 是       | 告警级别           | `CRITICAL`                 |
| `data.items[].status`           | string  | 是       | 告警状态           | `ACTIVE`                   |
| `data.items[].summary`          | string  | 是       | 告警摘要           | `温度45.2℃`                |
| `data.items[].triggerCount`     | integer | 是       | 触发次数           | `3`                        |
| `data.items[].firstTriggerTime` | string  | 是       | 首次触发时间       | `2026-02-01T15:05:35Z`     |
| `data.items[].lastTriggerTime`  | string  | 是       | 最后一次触发时间   | `2026-02-01T15:05:35Z`     |
